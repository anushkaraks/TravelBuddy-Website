# TravelBuddy Website

TravelBuddy Website is a web application designed to help users plan and organize their travel experiences. The project aims to provide useful features such as discovering destinations, creating itineraries, booking accommodations, and social interaction among travelers.

## Features

- Destination discovery
- Itinerary planner
- Accommodation booking
- Travel recommendations
- User authentication and profiles
- Community features (reviews, sharing, forums)

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) or relevant backend runtime
- [npm](https://www.npmjs.com/) or other package manager

### Installation

Clone the repository:
```bash
git clone https://github.com/anushkaraks/TravelBuddy-Website.git
cd TravelBuddy-Website
```

Install dependencies:
```bash
npm install
```

Configure environment variables as required.

### Running the Application

```bash
npm start
```
or use the appropriate command for your setup.
